# บันทึกการแก้โจทย์

ไฟล์นี้ต้องเขียนโดยนักศึกษาด้วยคำพูดของตนเอง

ใช้ template นี้เฉพาะกับโจทย์ OJ ที่ถูกระบุว่าต้องส่ง learning log เท่านั้น

ห้ามให้ AI เขียนไฟล์นี้แทนคุณ AI อาจช่วยตรวจ grammar, formatting หรือความชัดเจนได้ หลังจากที่คุณเขียนเนื้อหาของตนเองแล้ว

ถ้าใช้ AI กับโจทย์ learning-log-required นี้ ต้องทำไฟล์ `ai_reflection.md` ด้วย

## 1. ข้อมูล OJ

หมายเลข/ชื่อโจทย์ OJ:

```text
OJ3011 - Colors
```

OJ submission ID ถ้ามีการส่งแล้ว:

```text
542920
```

สถานะ OJ:

```text
Pass
```

เวลาที่ใช้คิดและทำโจทย์ด้วยตนเอง:

```text
30-60 minutes
```

เลือกหนึ่งข้อ:

```text
0-15 minutes
15-30 minutes
30-60 minutes
1-3 hours
3-6 hours
6-24 hours
1-3 days
4-7 days
1-4 weeks
More than 4 weeks
```

วิธีนับเวลา:

- นับเฉพาะเวลาที่ตั้งใจทำโจทย์นี้ด้วยตนเองจริง ๆ
- เริ่มนับตั้งแต่ตอนที่อ่านโจทย์ครั้งแรก
- ไม่นับเวลาพัก กินข้าว เรียน นอน เวลาที่ทำโจทย์อื่น หรือเวลาที่ไม่ได้ทำโจทย์นี้
- ถ้าใช้ AI ให้นับเฉพาะเวลาที่ทำด้วยตนเองก่อน prompt แรกที่ถาม AI
- ถ้าถามเพื่อน TA หรือผู้สอน ให้นับเฉพาะเวลาที่ทำด้วยตนเองก่อนขอความช่วยเหลือครั้งแรก
- ถ้าใช้ทั้ง AI และความช่วยเหลือจากคน ให้นับเฉพาะเวลาที่ทำด้วยตนเองก่อนขอความช่วยเหลือจากภายนอกครั้งแรก ไม่ว่าจะเป็น AI หรือคน
- ถ้าไม่ได้ใช้ AI และไม่ได้ขอความช่วยเหลือจากคน ให้นับเวลาถึงก่อนเขียน `submission.md`
- ประมาณเวลาได้ แต่ต้องซื่อสัตย์

## 2. ความเข้าใจโจทย์ของฉัน

เขียนโจทย์ด้วยคำพูดของตนเอง

ให้อธิบาย input, output และ constraints สำคัญด้วย

ถ้ายังไม่เข้าใจโจทย์ทั้งหมด ให้เขียนสิ่งที่เข้าใจในตอนนี้ ความเข้าใจอาจยังไม่ครบหรืออาจผิดได้ แต่ต้องพยายามอธิบายอย่างจริงใจ

```text
โจทย์มีเเม่สี 3 สี คือสีเเดง สีเหลือง สีน้ำเงิน เราสามารถนำแม่สีมาผสมกันได้ โดยมีเงื่อนไขคือ

หากเอาสีแดงผสมกับสีเหลือง จะได้สีส้ม
หากเอาสีแดงผสมกับสีน้ำเงิน จะได้สีม่วง
หากเอาสีเหลืองผสมกับสีน้ำเงิน จะได้สีเขียว

จากเงื่อนไขข้างบนให้เขียนโปรเเกรมผสมสีโดยรับเเม่สีเข้ามาทาง input 2 สี

Input :
2 บรรทัด เป็นชื่อสี

Output :
เป็นสีที่ผสมออกมาได้ หรือ เป็น Error หากเข้าเงื่อนไข

Constraints :
input เข้ามาเป็น Str เท่านั้น
```

## 3. แผนแรกของฉัน

เขียนแผนแรกก่อนรับความช่วยเหลือจาก AI เพื่อน TA ผู้สอน หรือก่อนสรุป code สุดท้าย

ถ้าใช้ AI ให้เขียนแผนที่มีอยู่ก่อน prompt แรกที่ถาม AI

ถ้าถามเพื่อน TA หรือผู้สอน ให้เขียนแผนที่มีอยู่ก่อนถาม

ถ้าไม่ได้ใช้ AI และไม่ได้ขอความช่วยเหลือจากคน ให้เขียนแผนที่มีอยู่ก่อนหรือระหว่างเริ่มเขียน code

แผนนี้เขียนแบบคร่าว ๆ ได้ อาจยังไม่สมบูรณ์หรืออาจต่างจากวิธีสุดท้าย

สามารถเขียนเป็น pseudocode, flowchart idea หรือขั้นตอนความคิดได้

```text
1. รับ input เข้ามาเป็นชื่อสี
2. สร้าง List สำหรับใส่ชื่อเเม่สีเอาไว้ ใน list มีสีน้ำเงิน สีเเดง เเละสีเหลือง
3. ใช้ if check ว่าสีทั้ง 2 สีที่ input เข้ามาเป็นเเม่สีไหม ถ้าไม่ print("Error")
4. ถ้าใช่ ก็ใช้ if ซ้อน if ในการ check ว่าถ้าสีเเรกเป็นสีเเดง เเล้วสีที่สองเป็นสีเหลือง ให้ print สีส้ม เเละใช้ or เขียนต่อว่าถ้าสีเเรกเป็นสีเหลือง เเล้วสีที่สองเป็นสีเเดงให้ print สีส้มเหมือนกัน เนื่องจาก or ถ้าเงื่อนไขฝั่งใดฝั่งนึงเป็นจริงเงื่อนไขนั้นก็จะเป็น True ทันที
5. เขียนเงื่อนไขการผสมสีให้ครบทุกสี 3 สี (ส้ม, ม่วง, เขียว) โดยเขียนเงื่อนไขเเบบบรรทัดที่ 4
6. เเสดงผล output ที่ได้ออกมา
```

## 4. วิธีสุดท้ายที่ใช้จริง

อธิบายสั้น ๆ ว่า algorithm หรือวิธีสุดท้ายที่ใช้จริงใน code ที่ส่งคืออะไร

หัวข้อนี้ต่างจาก Section 3:

- Section 3 คือแผนแรกก่อนใช้ AI ก่อนรับความช่วยเหลือจากคน หรือก่อนเขียน code สุดท้าย
- Section 4 คือวิธีสุดท้ายที่ใช้ใน solution จริง
- ถ้าวิธีสุดท้ายเหมือนกับแผนแรก ให้เขียนว่าเหมือนกัน และอธิบายสั้น ๆ ว่าทำไม

ห้ามคัดลอกคำอธิบายจาก AI

ห้ามคัดลอกคำอธิบายจากคนอื่น

```text
เนื่องจากเเผนเเรกของผม ผมคิดว่ามีการใช้ if-else ที่เยอะเกินไป อาจจะทำให้ติด pep-8 หรือ code ดูอ่านยาก เลยใช้ความรู้เรื่อง dict ที่เคยได้เรียนตอนค่าย prepro เข้ามาช่วยเเก้โจทย์

1. รับ input เข้ามาเป็นชื่อสี
2. สร้าง List สำหรับใส่ชื่อเเม่สีเอาไว้ ใน list มีสีน้ำเงิน สีเเดง เเละสีเหลือง
3. สร้างตัวเเปรเก็บค่าเเบบ dict ซึ่งกำหนด key คือ สองสีที่จะผสมกัน เเละ value คือ สีที่ผสมได้ ex. mix = {"RedYellow":"Orange"}
4. ใช้ if check ว่าสองสีที่ input เข้ามาเป็นเเม่สีไหม โดยการ if เช็คใน list
5. ถ้าสีที่ input เข้ามาเป็นเเม่สีทั้งคู่ให้ check ต่อเป็น if ซ้่อน if ว่า สองสีที่ input เข้ามาเป็นสีเดียวกันไหม ถ้าเป็นสีเดียวกันให้เเสดงผลสีนั้นออกมาเลย ex. สีเเรกที่ input เข้ามาคือเเดง สีที่สองก็เเดงเหมือนกัน ให้ print เเดงออกมาเลย เเละถ้าสีสองสีเป็นเเม่สีทั้งคู่ เเล้วทั้งสองสีไม่ใช่สีเดียวกันให้ join string สีที่ input สีเเรก กับสีที่สองเข้าด้วยกันเเล้วใช้ function sorted ในการเรียงลำดับอักษรภาษาอังกฤษ (A-Z) เเล้ว if เช็คอีกว่าข้อความที่ join เข้าด้วยกันเเล้วอยู่ในตัวเเปรที่เก็บข้อมูลเเบบ dict ไหม ถ้าอยู่เเล้ว print  value ของ key นั้นออกมา ถ้าไม่ print error
```


- อธิบายความหมายของโจทย์
- อธิบาย concept การเขียนโปรแกรม
- ให้ hint เกี่ยวกับแนวทาง
- คุยเรื่อง debugging
- คุยเรื่อง test cases
- ช่วยอธิบาย error message

สิ่งที่ไม่อนุญาต:

- คัดลอก code ของผู้อื่น

- ใช้ OJ submission ของผู้อื่น
- ขอให้ผู้อื่นส่ง OJ แทน
